PROJECT DESCRIPTION:
==================
Air Quality Index (AQI) Analysis and Forecasting System for Baguio City
- Processes air pollution data from EMB-CEPMO dataset
- Calculates AQI using US EPA standards
- Provides forecasting using Prophet time series model
- Generates visualizations and reports

NOTEBOOK CONTENTS (aqicolab.ipynb):
===================================

CELL 0: Import Libraries
- numpy, pandas, matplotlib, seaborn
- warnings configuration

CELL 1: Data Loading
- Loads EMB-CEPMOo1 year.csv dataset
- 24,344 rows of air quality data for Baguio City (2023)
- Columns: City, Date, PM2.5, PM10, NO, NO2, NOx, NH3, CO, SO2, O3, Benzene, Toluene, Xylene, AQI, AQI_Bucket

CELL 2: AQI Sub-Index Calculations
- Implements US EPA AQI calculation methods for:
  * PM10 Sub-Index
  * PM2.5 Sub-Index  
  * SO2 Sub-Index (1-hour average)
  * NOx Sub-Index (1-hour average)
  * NH3 Sub-Index (no official US EPA value)
  * CO Sub-Index (8-hour average)
  * O3 Sub-Index (8-hour average)
- Adds sub-index columns to dataframe

CELL 3: AQI Calculation
- Calculates final AQI as maximum of all pollutant sub-indices
- Fills NaN values in AQI column

CELL 4: AQI Bucket Classification
- Categorizes AQI values into:
  * Good (≤50)
  * Moderate (51-100)
  * Unhealthy for Sensitive Groups (101-150)
  * Unhealthy (151-200)
  * Very Unhealthy (201-300)
  * Hazardous (>300)

CELL 5: AQI Bucket Visualization
- Displays AQI bucket classification image

CELL 6: Prophet Model Setup
- Prepares data for time series forecasting
- Uses Prophet library with custom parameters:
  * holidays_prior_scale=0
  * seasonality_prior_scale=20
  * n_changepoints=50

CELL 7: Future Dataframe Creation
- Creates 31-day forecast period

CELL 8: Forecast Prediction
- Generates AQI predictions with confidence intervals

CELL 9: Model Validation
- Performs cross-validation
- Calculates accuracy: 86.67%

CELL 10: Forecast Visualization
- Creates interactive plotly visualization of predictions

CELL 11: Component Analysis
- Shows trend, seasonal components

CELL 12: Export Predicted Data
- Saves forecast to 'predicted_aqi_forecast.csv'

CELL 13: Export Historical Data
- Saves processed data to 'updated_air_quality_data.csv'

CELL 14: Package Installation
- Installs plotly and pandas

CELL 15: Historical Data Visualization
- Creates scatter plot of AQI over time with color-coded buckets

CELL 16: Trend Visualization
- Creates line plot of predicted AQI trend
