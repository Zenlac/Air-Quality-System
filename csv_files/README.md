# CSV Files Organization

This folder contains all CSV files used by the Air Pollution Forecasting System, organized by type.

## 📁 **Folder Structure**

### **data_files/**
Contains input data files for the system:

#### **sample_air_quality_data.csv**
- **Purpose**: Sample data file in strict format
- **Size**: 89.7 KB
- **Format**: 14 columns (Timestamp, AQI, 12 pollutants)
- **Usage**: Template for data format and system testing
- **Rows**: 365 days of sample data
- **Date Range**: 2023-01-01 to 2023-12-31

#### **EMB-CEPMOo1-year.csv**
- **Purpose**: Original EMB format data (for reference)
- **Size**: 2.17 MB
- **Format**: EMB format with 'Date' column
- **Usage**: Reference for EMB data conversion
- **Note**: Cannot be used directly in system (wrong format)

#### **EMB_converted.csv**
- **Purpose**: EMB data converted to strict format
- **Size**: 2.13 MB
- **Format**: 14 columns (Timestamp, AQI, 12 pollutants)
- **Usage**: Converted EMB data for system use
- **Source**: Converted from EMB-CEPMOo1-year.csv
- **Created**: Using convert_emb_data.py script

---

### **forecast_results/**
Contains forecast output files generated by the system:

#### **Forecast Results Files** (7 files)
- **Pattern**: `forecast_results_YYYYMMDD_HHMMSS.csv`
- **Purpose**: System-generated forecast results
- **Format**: Complete forecast data with all columns
- **Contents**: 
  - Forecasted AQI values
  - Confidence intervals (Upper/Lower Bound)
  - Prophet and ARIMA forecasts
  - AQI categories and health recommendations

## 📋 **Usage Guidelines**

### **For Data Input**
1. **Use sample_air_quality_data.csv** as format template
2. **Convert EMB data** before using in system
3. **Ensure strict format** compliance (14 exact columns)
4. **Validate data** before uploading to web interface

### **For Forecast Results**
1. **Files are auto-generated** by the system
2. **Download via web interface** in Forecast tab
3. **Export formats**: CSV, JSON, TXT available
4. **Timestamped filenames** for organization

### **File Formats**

#### **Input Data Format (Strict)**
```csv
Timestamp,AQI,PM2.5,PM10,NO2,CO,NO,NOx,NH3,SO2,O3,Benzene,Toluene,Xylene
2023-01-01,85,35.2,45.1,28.3,0.8,12.5,65.2,15.3,8.7,22.1,5.4,3.2
```

#### **Forecast Output Format**
```csv
Timestamp,Forecasted_AQI,Lower_Bound,Upper_Bound,Prophet_Forecast,ARIMA_Forecast,AQI_Category,Health_Recommendation
2023-12-02,87.5,75.2,99.8,88.2,86.8,Moderate,Sensitive groups should limit prolonged outdoor exertion
```

## 🔄 **Data Conversion**

### **EMB Data Conversion**
```bash
# Convert EMB format to strict format
python convert_emb_data.py --input EMB-CEPMOo1-year.csv --output EMB_converted.csv
```

### **Format Validation**
```bash
# Validate data format
python -c "
from src.data_processor import DataProcessor
from src.config import Config
processor = DataProcessor(Config('config.yaml'))
try:
    data = processor.load_data('csv_files/data_files/sample_air_quality_data.csv')
    print('✅ Data validation passed')
except Exception as e:
    print(f'❌ Validation failed: {e}')
"
```
